# GoCliente - Plataforma todo-en-uno con IA

![GoCliente](https://via.placeholder.com/1200x400?text=GoCliente+Website)

Sitio web oficial de **GoCliente**, una solución integral para agencias, consultores y negocios locales que buscan automatizar la captación de leads, la comunicación y las ventas mediante inteligencia artificial.

## 🚀 Características

- Diseño moderno y responsive
- Secciones completas: Hero, Estadísticas, Características, Testimonios, Precios, Formulario de captura
- Contadores animados
- Integración con imágenes gratuitas de Unsplash
- Formulario de demo funcional
- Menú móvil interactivo

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, Grid, animaciones)
- JavaScript (Intersection Observer, smooth scroll)
- Font Awesome 6
- Google Fonts (Inter)
